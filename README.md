# HexForm3D static GitHub site

Upload the contents of this folder to the root of your GitHub Pages repository.

The repository root should contain:
- index.html
- CNAME
- .nojekyll
- 404.html
- shop/
- licensed/
- originals/
- custom/
- cosplay/
- suppliers/
- gallery/
- about/
- contact/

Do not upload the ZIP file itself. Do not upload only one subfolder.

All customer checkout links intentionally go to the Squarespace shop at https://www.hexform3d.com so Squarespace continues handling inventory, cart, and checkout.
