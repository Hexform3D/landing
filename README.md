# HexForm3D static GitHub site

Upload the contents of this folder to the root of your GitHub Pages repository.

The repository root should contain:
- index.html
- CNAME
- .nojekyll
- 404.html
- shop/
- licensed/
- originals/
- custom/
- cosplay/
- suppliers/
- gallery/
- about/
- contact/

Do not upload the ZIP file itself. Do not upload only one subfolder.

All customer checkout links intentionally go to the Squarespace shop at https://www.hexform3d.com so Squarespace continues handling inventory, cart, and checkout.


## Brand color and logo update

This v5 package restores the original HexForm3D cube logo from the working baseline and includes four logo asset variants:

- `assets/hexform3d-logo-black.png`
- `assets/hexform3d-logo-red.png`
- `assets/hexform3d-logo-gray.png`
- `assets/hexform3d-logo-white.png`

The visible site uses the black logo on light headers and the white logo on dark footers. Accent colors are limited to the HexForm3D black/red/gray/white family; prior supplier logo-card green/pink styling has been removed.


V6 update: balanced the licensed-designer cards so Thin Air 3D matches the visual scale of Flexi Factory and Cutie3D, while keeping the black/red/gray/white HexForm3D palette.
